2D Semiconductor Database
================================= 

The database contains structural, elastic, electronic, thermodynamic, and optical properties of around 300 two-dimensional semiconductors. The properties are calculated by using the `VASP <https://vasp.at/>`_ code based on a semi-automated high-throughput screening workflow as implemented into the `VASPKIT <http://vaspkit.com/>`_ code together with bash shell. The Ô¨Ågures were generated by the `Matplotlib <https://matplotlib.org/>`_ code. The workflow for the calculation of the different properties is provided below.

[1] V. Wang, Y.-Y. Liang, Y. Kawazeo, W.-T. Geng, High-Throughput Computational Screening of Two-Dimensional Semiconductors, `arXiv:1806.04285. <https://arxiv.org/abs/1806.04285>`_
